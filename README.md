# Laura Viale - Personal Website

This is the personal website for Laura Viale, a linguist, educator, and AI localization specialist. The site is built using the [Vonge Jekyll template](https://github.com/CloudCannon/vonge-jekyll-bookshop-template) and has been customized and modified for deployment to Netlify.

![Laura Viale website screenshot](_screenshot.png)

## Template Origin

This site is based on the [Vonge template](https://github.com/CloudCannon/vonge-jekyll-bookshop-template) created by [CloudCannon](http://cloudcannon.com/). The Vonge template is a personal portfolio/blog site template for Jekyll with a component library built for use with [Bookshop](https://github.com/cloudcannon/bookshop/).

## Customization

This version has been modified from the original Vonge template to:
- Work with Netlify deployment
- Feature Laura Viale's professional content and portfolio
- Include custom pages for Courses and Contact
- Reflect Laura's specific branding and services

## Development

This site is built with [Jekyll](http://jekyllrb.com/) and uses the Bookshop component library.

Install the dependencies:

~~~bash
$ npm install
~~~

Install the Jekyll dependencies with [Bundler](http://bundler.io/):

~~~bash
$ npm run jekyll:install
~~~

Run the website locally:

~~~bash
$ npm start
~~~

## Deployment

This site is configured for deployment to Netlify via GitHub. The `netlify.toml` file contains the build configuration.
